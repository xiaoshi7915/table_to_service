# AI模型Logo存储目录

## 目录说明

此目录用于存储各个AI模型提供商的Logo图片文件。

## 文件命名规则

Logo文件应按照以下规则命名：

- 文件名格式：`{provider}.png`
- 支持的格式：`.png`、`.jpg`、`.svg`（推荐使用PNG格式，透明背景）
- 文件大小：建议不超过200KB
- 图片尺寸：建议 200x200 像素或更高（保持1:1比例）

## 使用说明

1. 下载对应提供商的Logo图片
2. 将图片重命名为对应的provider名称（如：`deepseek.png`）
3. 上传到此目录（`/opt/table_to_service/frontend/public/images/logos/`）
4. 刷新前端页面即可看到Logo显示

## Logo要求

- **格式**：PNG（推荐，支持透明背景）或 SVG
- **尺寸**：200x200 像素或更高，保持1:1比例
- **背景**：透明背景（PNG）或白色背景
- **质量**：清晰，无模糊或锯齿
- **文件大小**：建议不超过200KB

## 注意事项

- Logo文件必须严格按照provider名称命名
- 如果Logo文件不存在，系统会自动显示默认图标
- 支持PNG、JPG、SVG格式，但推荐使用PNG格式以获得最佳显示效果
